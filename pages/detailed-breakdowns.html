Project Breakdowns

This page lists all of my past and active projects, going into detail about their ins and outs

BoatIt is a competitive boating game where you are tasked with picking up a limited amount trash from a body of water.

PowerIt is a take on a typical wave-based tower defense game. The user has to juggle a 'power cube' to either power their defenses or collect coins which are used for power-ups!

HauntIt follows the tale of a grave-keeping wizard! Use an array of spells to ward off unwanted spirits and keep your graveyard safe

BoatIt

  • Target Platform: PC & Web GL
  • Tools: Unity Engine 5, Blender, C#, GitHub
  • Duration: 1 Week
  • Team Size: 1

  • Leaderboards - While this is kept local on a per-game basis, it can be easily extended to include an online scoreboard

  • PlayerPrefs - Primarily used to save the volume of the music, can be extended to cover custiomization of the boat

  • EventSystem - A decoupled event system based on the ScriptableObject structure in Unity

BoatIt

  • Target Platform: PC & Web GL
  • Tools: Unity Engine 5, Blender, C#, GitHub
  • Duration: 2 Weeks
  • Team Size: 1

  • Wave Generator - The enemy wave system utilises abstract classes, as a result the waves can be modified to include 'x' amount of 'y' enemies on the fly

  • Enemy Types - For variety, the 'game' features 3 types of enemies: Red grunts, Yellow Sprinters and Bosses. Red grunts target the turret, where as yellow ones go after the base tower!

BoatIt

  • Target Platform: PC & Console
  • Tools: Unity Engine 5, Blender, C#, GitHub, Audacity
  • Duration: 1 Month
  • Team Size: 1

  • Decoupled Stat and Event Systems - A lot of the project focused on having dynamic systems which allow developers to create spells with ease, this was extended to the attributs of the player and NPC/Enemies which are scalable

  • Controller Support - The gameplay was designed to support spell-casting from controllers! With the big picture of making it a couch co-op game such as Overcooked!

  • Spell Casting - The user has access to spells as well as their trusted wand which can be used for melee
