Implementing the AI for this project was a fun challenge. Usually, I'd use a NavMeshAgent to create my AI behaviours, and set a target destination using SetDestination(Vector3 target)

But, because I wanted to implement dynamic water physics, with rigidbodies and waves and a physics-based boat controller with drag, I figured a pathing mesh would be difficult to create for a dynamic water body.

The solution was to set the directions using dot product for movement, and a cross product to apply a directional force (left/right) toward the target object
